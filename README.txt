== Description ==

= MarfeelPress: The completely-free monetization plugin =

Marfeel is the platform used to optimize the revenue of up to a billion mobile sessions per month. MarfeelPress is the all-in-one plugin that brings that power to WordPress publishers.

Publishers such as euronews or The South African have used Marfeel to monetize their blogs resulting in revenue increases of up to 40% alongside perfect mobile experiences.

From page speed, to new sources of traffic, MarfeelPress does everything to generate a fully-optimized mobile page that comes connected to every major ad network and Marfeel’s ad-serving technology.

MarfeelPress helps publishers optimize their metrics through our all-in-one solution:


= REVENUE =
* One-point connections to premium monetization networks
* Server to server header-bidding
* AI-powered ad-dealer for optimized ad placements
* Real-time bidding

Our team continues to develop tools to optimize the frequency, position, and density of ads for better results. Assets are lazy-loaded so your monetization loads instantly and doesn’t affect your content performance.

= TRAFFIC =
* Google AMP integration
* SEO performance tools
* Analytics integration

Connect new channels to your page such as Google AMP, Apple News, and Facebook Instant Articles. Boost your page ranking with SEO tools that work in conjunction with Yoast, page layouts that are proven to increase page views per reader, and integrates with your favorite analytics tools.

= SPEED =
* Fully-optimized Progressive Web App (PWA)
* Lazy-loaded assets
* Server-side device detection

Don’t settle for responsive design, create your own PWA for an app-like experience that loads almost instantly. By prioritizing primary content, utilizing server-side device detection, and lazy-loading assets, you’ll cut your bounce rate and keep your readers on the page for longer.

= ENGAGEMENT =
* Automatic & manual push notifications
* One-finger swipe between articles and sections
* Recirculation and retention tools

Wondered how the biggest publishers are able to engage their readers so well? Apply designs and layouts that engage with users to increase pageviews and send push notifications to re-engage offline users.

= FULL SUPPORT =
As well as full email support throughout, our team offers free live chat assistance to guide you through the installation and setup. We then offer continued support to ensure your revenue is consistently higher with MarfeelPress.

== Installation ==
= INSTALL FROM WORDPRESS =
1. Visit ‘Plugins > Add New’
2. Search for ‘MarfeelPress”
3. Activate it from your Plugins page.
4. Open the plugin and start customizing your mobile site.

Bonus: For any questions, there is a chat inside the plugin where you can ask our team for help

= MANUAL INSTALLATION =
1. Download the .zip file from this page
2. Enter your WordPress admin page
3. Visit Plugins > Add New > Upload Plugin and select the .zip file
4. Go to your plugins page and activate MarfeelPress
5. Enter the plugin and start customizing your mobile site.
